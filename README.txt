IMMOHOESSEN-SITE — Como publicar (resumo)

1) Suba TODOS os arquivos desta pasta para seu repositório no GitHub (raiz).
2) Em Settings → Pages → Deploy from a branch → Branch: main / (root) → Save.
3) Troque img/cover.jpg por uma foto sua (mesmo nome).
4) Em offre.html, troque ACTION_AQUI pelo seu link do Formspree.
5) Depois de publicado, edite sitemap.xml e robots.txt com a URL final do site.
